# Celestial Tic-Tac-Toe

A premium Tic-Tac-Toe web application built using HTML, CSS, and JavaScript. This project enhances the classic game with intelligent AI opponents, multiple visual themes, sound effects, score persistence, animations, and a fully responsive modern user interface.

## Overview

Celestial Tic-Tac-Toe provides an immersive gaming experience through advanced gameplay mechanics, polished UI design, AI-powered opponents, and interactive visual effects. The application is designed to be responsive across desktop, tablet, and mobile devices.

## How to Run

### Method 1: Browser

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The game will launch in your default browser.

### Method 2: Visual Studio Code

1. Open the project folder in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
5. The application will launch automatically.

### Method 3: Native macOS Application

1. Use the included `build_mac_app.sh` script.
2. Build the native macOS application.
3. Launch the generated application directly from Finder.

## Gameplay

### Player vs Player

- Enter player names.
- Alternate turns between X and O.
- First player to align three symbols wins.

### Player vs AI

- Select a difficulty level.
- Challenge the AI opponent.
- Test your skills against the Invincible Minimax AI.
